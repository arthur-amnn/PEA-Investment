2 Rituel mensuel (~10 min)

À date fixe — ex. le 1er du mois
  1. Ouvre l'app → Dashboard → repère « Prochain ordre » (ex. C2 · PAEEM · 180 €).
  2. Vire l'argent sur le PEA Fortuneo = DCA du mois + résiduel reporté (le résiduel est affiché dans Suivi DCA → Compte espèces). Attends 24–48 h que les fonds soient disponibles.
  3. Sur Fortuneo : 1 seul ordre sur l'ETF du cycle → achète le maximum de parts entières possible (gratuit si < 500 €). Note le nb de parts et le prix.
  4. Enregistre dans l'app (Suivi DCA) : a) Compte espèces → + virement (montant + date) ; b) Formulaire d'ordre → parts, prix, date → Enregistrer. Le résiduel se recalcule tout seul (virements − achats).
  5. (option) Dashboard → saisis les cours du jour pour voir ta plus-value.
  6. Coche la checklist mensuelle (onglet Checklists).
⭐ Règle d'or : ne jamais sauter un mois, et toujours enregistrer le virement ET l'ordre.

3 Rendez-vous périodiques

À ne pas oublier
  • Tous les 6 mois (Juin & Décembre) — onglet Bilan : saisis les valeurs de marché → l'app donne l'écart vs 60/30/10 et l'action (rien si < 5 pts · surpondérer si 5–10 pts · arbitrer si > 10 pts). Puis Export → Télécharger JSON (sauvegarde).
  • 2031 — passe le DCA de 180 € → 350 € (l'app affiche un bandeau « Passer à 350 € » → clique).
  • 2036 — passe à 500 € (Scénario B) (idem, l'app te le propose).
  • ~2055 — le plafond 150 k€ de versements approche (Sc. B) → ouvre un CTO pour continuer (le PEA reste).

4 Cas particuliers

  • Bonus, prime, vente, parrainage → Suivi DCA → bouton « + Apport exceptionnel » → enregistre-le en C0 (n'avance pas le cycle, compté daté dans la courbe « Réel »). Enregistre aussi le virement correspondant.
⚠️ Krach / chute des marchés → bouton Mode Crise : règle des 48 h, ne JAMAIS vendre, continuer (voire doubler) le DCA. Sur 30 ans, le plan absorbe les crises.

5 Pérennité 30 ans

Le fichier = ton trésor · le JSON = tes données
  • Garde PEA_Investment.html en ≥ 2 endroits (OneDrive + une 2ᵉ copie : disque externe / autre cloud / mail à toi-même). Il est autonome → marche toujours, hors-ligne.
  • Exporte le JSON au moins 2×/an (aux bilans) et garde les derniers fichiers.
  • Tu changes de téléphone → ré-ouvre l'URL dans Safari → « Sur l'écran d'accueil » → puis Export → Importer ton dernier JSON.
  • L'hébergeur ferme → reprends le fichier, re-dépose-le ailleurs (GitHub Pages, Cloudflare Pages…) → nouvelle URL → Importer le JSON.
  • Gros problème → le dossier DOC/ (kit + 216 tests) permet de tout réparer (le donner à un dev ou une IA).
